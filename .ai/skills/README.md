# Shared skills

The skill directories in this folder are the canonical project-level skills.
Each skill uses the Agent Skills layout:

```text
.ai/skills/<skill-name>/SKILL.md
```

The following tool directories are symlinks to this folder:

- `.claude/skills/`
- `.cursor/skills/`
- `.opencode/skills/`
- `.codex/skills/`

Keep skill instructions and supporting files under `.ai/skills/`; do not edit a
tool-specific link separately.
